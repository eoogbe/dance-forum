# Dance Forum

**WORK IN PROGRESS**

A example forum so I can learn [Laravel](http://laravel.com).

## Features

- User roles
- Admin panel
- WYSIWYG for formatted text
- Categories
  - CRUD operations (admin)
  - order (admin)
- Boards
  - CRUD operations (admin)
  - order (admin)
  - user view count
- Topics
  - create
  - edit (admin)
  - delete (admin)
  - pagination
  - user view count
  - pin (admin)
- Posts
  - create
  - edit (author or admin)
  - soft delete (author or admin)
  - restore deleted posts (admin)
  - pagination
