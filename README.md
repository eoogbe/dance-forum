# Dance Forum

**WORK IN PROGRESS**

A example forum so I can learn [Laravel](http://laravel.com).

## Features

- Admin panel
- WYSIWYG for formatted text
- pagination
- Users
  - assign roles (admin)
  - edit settings and profile
  - cancel account
- User roles
  - CRUD operations (admin)
  - assign users (admin)
- Categories
  - CRUD operations (admin)
  - order (admin)
  - permissions (admin)
- Boards
  - CRUD operations (admin)
  - order (admin)
  - user view count
  - permissions (admin)
- Topics
  - create
  - edit (admin)
  - delete (admin)
  - user view count
  - pin (admin)
  - lock (admin)
  - hide (admin)
  - permissions (admin)
  - subscriptions
- Posts
  - create
  - edit (author or admin)
  - soft delete (author or admin)
  - restore deleted posts (admin)
